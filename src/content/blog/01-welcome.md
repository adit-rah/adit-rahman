Welcome to my personal terminal interface. This website is built to feel like you're accessing a Vault-Tec terminal from the Fallout universe, but with all the modern conveniences of a React application.

### What You'll Find Here

- **ABOUT**: Learn about my background and experience

- **CONTACT**: Get in touch with me

### Navigation

Type `help` to see all available commands, or simply type the section name you want to visit (e.g., `about`, `projects`).

### Technical Notes

This site is built with:
- React + TypeScript for the frontend
- React Router for navigation
- Tailwind CSS for styling
- Vite for build tooling
- GitHub Pages for hosting

The terminal interface is fully functional - try typing commands!
